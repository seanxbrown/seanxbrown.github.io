React Quotes Application

Live Site Github Repo

This application allows users to either search for a quote by a specific person or receive a random quote. The user can also save the quote to a list of saved quotes (saved in local storage and displayed on a new page using React Router). Quotes can also be removed after they're saved. New additions include a page with details of the top 150 authors with the most quotes.

Technologies Used:

  • CSS
  • Javascript (inc API calls using fetch, data saved in localstorage)
  • React Router
  • React Hooks (state and effect)
  • React Bootstrap
  • Git
