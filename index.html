React Weather Application

Live Site Github Repo

Application that consumes the OpenWeather API. The user can enter a city and receive the current weather plus an 8-day forecast. The user input is used in the API call, and the response is displayed using components, props and state.

Technologies Used:

  • Javascript
  • React (including hooks, state & fetch)
  • React Bootstrap
  • Git

React Brewdog Application

Live Site Github Repo

Application that consumes the Brewdog API. The API is called when the page loads and the result of the API call (list of beers) is stored in state. The beers are displayed and filtered using the state data. This improves performance time vs calling the API with each new request.

Technologies Used:

  • Javascript
  • React (including hooks, state & fetch)
  • React Bootstrap
  • Git
